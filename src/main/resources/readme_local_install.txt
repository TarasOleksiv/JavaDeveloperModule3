Структура папки "resources".
    1) config.properties - файл конфігурації проекту:
        Файл містить основні налаштування для конекту до сервера mysql та деякі інші дані.
        ПЕРЕД ЗАПУСКОМ ПРОГРАМИ НЕОБХІДНО ПРОПИСАТИ В НЬОМУ ПРАВИЛЬНІ ЗНАЧЕННЯ ДЛЯ ЮЗЕРА ТА ПАРОЛЮ НА СВІЙ СЕРВЕР!
    2) DBscheme.png - схема бази.
    3) initDB.sql - створює всі необхідні таблиці на базі.
    4) populateDB.sql - заповнює даними таблиці.
    5) readme_local_install.txt - файл з інструкцією.
_____________________________________________________

Схема запуску програми на локальній машині.
    1) ПЕРЕД ЗАПУСКОМ ПРОГРАМИ НЕОБХІДНО ПРОПИСАТИ у файлі resources\config.properties ПРАВИЛЬНІ ЗНАЧЕННЯ ДЛЯ ЮЗЕРА ТА ПАРОЛЮ НА СВІЙ СЕРВЕР!
    2) Запустити програму.
        При запуску програма перевіряє чи на серевері mysql існує база. прописана у конфіг-файлі з пункту 1;
            Якщо база існує. то програма продовжує роботу,
            якщо база не існує, то програма автоматично запускає на виконання 2 скрипти з папки resources (initDB.sql, populateDB.sql):
                ці скрипти створюють на сервері базу, таблиці та заливають дані в таблиці.
    3) Продовжити роботи в Головному меню.
